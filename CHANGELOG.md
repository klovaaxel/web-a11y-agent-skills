# Changelog

## 0.1.0 - 2026-04-26

- Added ten portable web accessibility skills covering authoring, review, testing, forms, navigation, dynamic UI, CSS, web components, debugging, and orchestration.
- Added five Cursor subagents for accessibility orchestration, implementation, review, remediation, and testing.
- Added Claude Code compatibility with Claude skill and subagent installers.
- Added GitHub Copilot compatibility with `.github/skills` and `.github/prompts` installers.
- Added OpenCode compatibility with OpenCode skill and subagent installers.
- Added trust assets: compatibility badges, skill list screenshot, example output, changelog, package metadata, and public attribution notes.
